# no-code-pj

## 📌 概要

**no-code-pj** は、フォーム作成・回答収集・管理を行うWebアプリケーションです。  
ユーザーがフォームを作成し、回答を受け取り、それを一覧・詳細・集計として管理できるシンプルな「ノーコードフォームシステム」を目指しています。

ポートフォリオ用途として、「1つの完成したWebアプリ」として見せられることを目的に開発しています。

---

## 🚀 主な機能

### ① フォーム作成
- 新しいフォームの作成
- フォームタイトル・構造の登録

### ② フォーム回答
- 作成されたフォームへの入力
- 回答データの送信・保存

### ③ 回答一覧表示
- フォームごとの回答一覧表示
- データの確認・管理

### ④ 回答詳細表示
- 個別回答の詳細閲覧

### ⑤ 管理機能（APIベース）
- データの保存（save.php）
- 更新（update.php）
- 削除（delete.php）
- 一覧取得（list.php）
- 詳細取得（detail.php）
- 回答送信（submit.php）
- エクスポート（export.php）

---

## 🧱 技術構成

### フロントエンド
- Next.js（App Router）
- TypeScript
- React

### バックエンド
- PHP
- SQLite（database.sqlite）

### 構成概要
- フロントエンド：UIと画面遷移
- バックエンド：API処理（CRUD）
- DB：軽量ローカルDB（SQLite）

---

## 📂 画面構成

- `/list`：フォーム一覧
- `/form/[id]`：フォーム入力画面
- `/answers/[id]`：回答一覧・詳細
- `/create`：フォーム作成画面

---

## 🧠 工夫した点

- フロントエンドとバックエンドを明確に分離
- PHP + SQLiteによる軽量なAPI構成
- フォーム単位でデータ管理できる設計
- シンプルな構成で拡張しやすい構造を意識

---

## 🎯 このプロジェクトの目的

このアプリは以下を目的として作成されています：

- ノーコード的なフォームシステムの実装
